# Tere tulemast Claude Code CMS-i

See on näidis blogipostitus, mis demonstreerib, kuidas markdown-põhine sisu süsteem töötab.

## Markdown tugi

Saad kasutada kõiki tavalisi markdown formaate:

- **Paks tekst** ja _kaldkiri_
- Nummerdatud ja täpploendid
- [Lingid](https://example.com)
- Pealkirjad (H1-H6)
- Koodiplokid

### Koodi näide

```javascript
function greet(name) {
  console.log(`Tere, ${name}!`)
}
```

## Pildid

Pildid võivad olla suhtelise teega (`./images/`) või absoluutse teega (`/images/blog/`).

## Järgmised sammud

1. Asenda see postitus oma sisuga
2. Lisa oma pildid `public/images/blog/` kausta
3. Uuenda frontmatter'i oma andmetega
